# 🛒 ShopHub

A modern **Full-Stack E-Commerce Web Application** built with **Next.js 16**, **TypeScript**, **Tailwind CSS**, **shadcn/ui**, and **NextAuth**.

ShopHub provides a complete online shopping experience with authentication, cart management, checkout, password recovery, dark mode, and responsive UI.

> 🚀 Live Demo: https://shop-hub-zrhu.vercel.app

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/MostafaMohamid/ShopHub.git
```

## Install Packages

```bash
npm install
```

## Environment Variables

Create a `.env.local`

```env
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000
```

## Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
